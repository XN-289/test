# PGA (Polyglycolic Acid)

## Description
A biodegradable, thermoplastic polymer and the simplest linear, aliphatic polyester. Used extensively in synthetic absorbable sutures.

## Hierarchy
- **Type**: [[Material]]

## Technical Properties
- **Absorption Method**: Hydrolysis.
- **Tensile Strength Retention**: High for the first 14-21 days.
- **Complete Absorption**: 60 to 90 days.
