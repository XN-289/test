## 🎨 品牌风格指南

### 语言风格

1. **专业但不晦涩**
   - ✓ 使用行业术语，但提供解释
   - ✗ 避免过度技术化

2. **自信但不傲慢**
   - ✓ 突出优势，提供数据支持
   - ✗ 避免贬低竞品

3. **简洁但不简单**
   - ✓ 重点突出，逻辑清晰
   - ✗ 避免冗长描述

4. **行动导向**
   - ✓ 明确的 CTA，引导用户行动
   - ✗ 避免模糊表述
