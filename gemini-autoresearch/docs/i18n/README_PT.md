> Transforme o Gemini CLI em um motor de melhoria autônomo implacável.
> Defina uma meta. Afaste-se. Acorde com os resultados.

Esta é uma competência para o 'Gemini CLI' e 'Antigravity IDE' que itera de forma autónoma em qualquer tarefa com um resultado mensurável.

---

## Como funciona (How it works)

1. **Meta (Goal)**: Descreva o que pretende melhorar.
2. **Escopo (Scope)**: Ficheiros ou pastas que a IA pode alterar.
3. **Métrica (Metric)**: O número que está a otimizar.
4. **Verificação (Verify)**: Comando que mede o progresso.
5. **Guarda (Guard)**: Comando que garante a estabilidade.
