> Превратите Gemini CLI в неутомимый механизм автономного улучшения.
> Поставьте цель. Уйдите. Проснитесь с результатами.

Это функция для 'Gemini CLI' и 'Antigravity IDE', которая автономно работает над любой задачей с измеримым результатом.

---

## Как это работает (How it works)

1. **Цель (Goal)**: Опишите, что вы хотите улучшить.
2. **Область (Scope)**: Файлы или папки, которые ИИ может изменять.
3. **Метрика (Metric)**: Число, которое вы оптимизируете.
4. **Проверка (Verify)**: Команда, измеряющая прогресс.
5. **Защита (Guard)**: Команда, обеспечивающая стабильность.
