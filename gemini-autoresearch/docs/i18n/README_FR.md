> Transformez Gemini CLI en un moteur d'amélioration autonome et acharné.
> Fixez un objectif. Partez. Réveillez-vous avec les résultats.

Ceci est une compétence pour 'Gemini CLI' et 'Antigravity IDE' qui permet d'itérer de manière autonome sur n'importe quelle tâche ayant un résultat mesurable.

---

## Comment ça marche (How it works)

1. **Objectif**: Décrivez ce que vous voulez améliorer.
2. **Champ d'application (Scope)**: Fichiers ou dossiers que l'IA peut modifier.
3. **Métrique (Metric)**: Le nombre que vous optimisez.
4. **Vérification (Verify)**: Commande qui mesure le progrès.
5. **Garde (Guard)**: Commande qui vérifie la stabilité.
