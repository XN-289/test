> 将 Gemini CLI 变成一个不懈的自主改进引擎。
> 设定目标。离开。醒来看到结果。

这是 'Gemini CLI' 和 'Antigravity IDE' 的一项技能，用于在任何具有可衡量结果的任务上进行自主迭代。

---

## 工作原理 (How it works)

1. **设定目标**: 描述你想要改进的内容。
2. **范围 (Scope)**: 指定允许更改的文件。
3. **指标 (Metric)**: 你想要优化的数字。
4. **验证 (Verify)**: 测量结果的命令。
5. **守卫 (Guard)**: 确保没有其他破坏的命令。

---
