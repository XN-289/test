1. 项目概览

ViaSurg AI CMO Platform — Executive Summary

系统是什么

ViaSurg AI CMO Platform 是一个医疗器械智能营销系统，通过 5 阶段自动化 Pipeline， 将市场数据转化为可发布的 SEO 优化页面。系统集成了 DeepSeek LLM、Google Keywords Planner 数据、 Agent SDK 等技术，实现从信号采集到质量验证的全流程自动化。

核心能力

📊 智能市场分析

自动采集竞争情报、关键词 CPC 数据、市场趋势，生成 OODA 分析报告。

📝 自动页面生成

EN/ZH 双语切换，i18n 数据驱动，所有生成页面均支持中英文。

3. 已完成工作

Delivered Features & Bug Fixes

核心功能交付

  • 5 阶段 Pipeline 全链路打通 — 信号采集 → 策略分析 → Wiki 编译 → 页面生成 → 质量验证，一键运行
  • 53 个 SEO 页面自动生成 — 覆盖 12 产品 + 10 竞品 + 8 分类 + 2 材质 + 1 技术 + 8 通用实体 + 首页
  • 86% 高价值关键词覆盖 — 373 个关键词中 319 个已匹配到对应页面，通过同义词/材质/分类多维度匹配
  • 关键词注入系统 — 每个页面自动生成 <meta keywords>、FAQ 问答、关键词标签侧边栏

  • Sitemap 生成 — 42 个 URL，按页面类型差异化优先级（产品 0.9 > 分类 0.8 > 其他 0.7）
  • i18n 双语支持 — EN/ZH 一键切换，所有 53 个页面均支持中英文
  • 实时 SSE 推送 — Pipeline 执行过程中实时推送进度和日志到前端控制台
  • 页面再生成功能 — 删除页面后可通过 Dashboard 一键检测缺失并重新生成，无需访问后台
  • OODA 分析修复 — Observe 信号数从 0 修复为正确值 4，数据流结构对齐

How to Start & Operate

# 进入项目目录 cd d:/test/kealin/demo # 启动服务器 node server.js # 系统将在 http://localhost:3001 启动

访问地址

页面地址用途

生成站点http://localhost:3001/output/index.html查看生成的 SEO 页面
APIhttp://localhost:3001/api/pipeline/run触发全流程

操作流程

一键运行全流程

在控制台点击 ▶ Run Pipeline 按钮，系统将依次执行 5 个阶段，实时显示进度和日志。

单独运行某个阶段

每个阶段（信号采集、策略分析、Wiki 编译、页面生成、质量验证）都有独立的运行按钮。

页面再生

如果页面被误删，进入 Site Generator 标签，点击 🔍 Check Pages 检测缺失，然后点击 🔄 Regenerate 一键恢复。

查看质量报告

运行 Verification 后查看质量评分、关键词覆盖率、Schema 覆盖等指标。

5. 关键词与 SEO 优化

Keyword Coverage & SEO Enhancement

关键词研究与覆盖

系统内置关键词扩展引擎，将 1271 个种子关键词通过同义词、分类、材质、品牌、手术方式等维度扩展至 3797 个。 同时支持接入 Google Ads API 获取真实搜索量和 CPC 数据。 扩展后的关键词通过多维度匹配算法关联到对应页面，覆盖率达 86%。

每个页面的 SEO 元素

  • <meta name="keywords"> — 自动注入相关高价值关键词
  • <meta name="description"> — 从 Wiki 实体内容提取的描述
  • Schema.org JSON-LD — Product / Organization / CollectionPage 等结构化数据
  • FAQ Section — 5 种问答角度，使用真实规格、竞品、监管数据
  • Keyword Tags 侧边栏 — 展示相关关键词及 CPC 出价

  • Sitemap — 含 lastmod 日期，差异化优先级
